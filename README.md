# CCNA Packet Tracer Labs

CCNAの学習過程でCisco Packet Tracerを用いて構築した演習ファイル集です。
ルーティング・スイッチング・セキュリティ・冗長化の基礎を、実際にパケットフローを動かしながら確認しました。
